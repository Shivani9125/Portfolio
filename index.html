Frequently Asked Questions

Basic questions about me, Laravel, and the kind of work I do.

Who is Shivani Dwivedi?

I am a PHP and Laravel developer with experience building web applications, REST APIs, admin panels, and reporting systems. I currently work at Inventics Software and have contributed to projects like JGH, Lux, Yoga, and Gaushala.

What is Laravel?

Laravel is a popular PHP framework used to build web applications and APIs quickly and securely. It provides tools for routing, database handling, authentication, and more — which is why I use it for most of my backend work.

Why do you use Laravel for development?

Laravel makes it easier to write clean, organized code. It speeds up API development, handles security basics well, and works great with MySQL. For projects like CRM panels, admin dashboards, and third-party integrations, Laravel is a reliable choice.

What Laravel skills do you have?

I work with Laravel MVC structure, REST API development, database migrations, authentication, form validation, third-party API integration, and admin panel development. I also optimize queries and maintain code that other developers can easily work with.

Do you build APIs for mobile apps?

Yes. I have built third-party APIs for projects like Yoga that mobile and Flutter teams integrate with. I focus on clear JSON responses, proper error handling, and endpoints that are easy for frontend developers to use.

What projects have you worked on?

JGH — reporting modules, Lux — CRM panel with Tattelle and third-party API integrations, Yoga — third-party REST APIs, and Gaushala — admin panel with backend API. These are real business applications, not demo websites.
